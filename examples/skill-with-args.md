# 引数を受け取るスキルのテンプレート

## 概要

引数を受け取って処理を行うスキルのテンプレートです。

## 使い方

```
/スキル名 <引数>
```

例:
```
/スキル名 example.txt
/スキル名 "複数の単語を含む引数"
```

## 引数

`$ARGUMENTS` で渡された引数を使用します。

- 引数が指定されている場合: その値を使って処理
- 引数がない場合: ユーザーに確認を求める

## 実行手順

1. $ARGUMENTS の内容を確認する
2. 引数がない場合はユーザーに入力を求める
3. 引数を使って処理を実行
4. 結果を表示

## 注意事項

- 引数が必須の場合は、ない場合の処理を明示的に書く
