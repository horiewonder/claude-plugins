# Generate README

## 概要

プロジェクトの構造を分析して、README.mdを自動生成します。

## 使い方

```
/generate-readme
```

## 実行手順

1. プロジェクトのルートディレクトリを確認
2. 以下の情報を収集:
   - package.json / pyproject.toml / Cargo.toml などの設定ファイル
   - ディレクトリ構造
   - 主要なソースファイル
   - 既存のドキュメント
3. 以下のセクションを含むREADMEを生成:
   - プロジェクト名と説明
   - インストール方法
   - 使い方
   - 主な機能
   - ディレクトリ構造
   - 開発環境のセットアップ
   - ライセンス
4. 生成したREADMEをユーザーに提示
5. 承認されたらファイルを作成/更新

## カスタマイズ

生成後、以下を手動で追加することを推奨:
- バッジ（CI/CD、カバレッジなど）
- スクリーンショット
- デモリンク

## 注意事項

- 既存のREADME.mdがある場合は上書き確認を行う
- プライベートな情報（APIキーなど）を含めないよう注意
