# Find and Replace

## 概要

プロジェクト全体で文字列の検索と置換を行います。

## 使い方

```
/find-replace <検索文字列> <置換文字列>
```

例:
```
/find-replace "oldFunction" "newFunction"
/find-replace "TODO" "DONE"
```

## 実行手順

1. $ARGUMENTS から検索文字列と置換文字列を取得
2. 引数が不足している場合はユーザーに確認
3. プロジェクト全体で検索文字列を検索（.gitignoreを考慮）
4. 検索結果を一覧表示:
   - ファイルパス
   - 該当行と行番号
   - 置換プレビュー
5. **ユーザーに確認を求める**（全置換/選択置換/キャンセル）
6. 承認されたら置換を実行
7. 変更されたファイルの一覧を表示

## 対象外ファイル

以下は検索対象から除外:
- `node_modules/`
- `.git/`
- バイナリファイル
- その他 .gitignore で指定されたファイル

## 注意事項

- 大量の置換を行う前に必ずgitでコミットしておくことを推奨
- 正規表現は現時点ではサポートしていない（リテラル検索のみ）
